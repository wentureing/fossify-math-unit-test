二、选定测试应用
应用名称：Fossify Calculator（开源计算器）
应用简介：一款功能全面、离线运行、高隐私安全的开源计算器工具，界面简约、功能完备，支持自
定义设置，全程无第三方数据收集，代码开源透明。
三、应用核心功能
1. 离线运行：无需网络权限，随时随地使用，隐私、稳定性更优
2. 多元计算：支持基础加减乘除、开方、乘方，及房贷等复杂计算
3. 自定义设置：按键振动、屏幕常亮、界面颜色个性化配置
4. 隐私保障：不收集、不共享任何用户个人信息
5. 计算历史：支持历史计算记录查询、回看
6. 开源透明：项目完全开源，可审计源码，安全可信
   四、应用初步编译流程
1. 源码获取：从 GitHub 官方仓库下载 Fossify Calculator 完整源码
2. 环境配置：搭建 Android Studio 编译环境，配置对应 SDK、Gradle 依赖
3. 项目导入：将源码导入开发工具，完成项目依赖同步
4. 编译构建：排查依赖问题，编译构建项目，生成可运行安装包
5. 运行验证：在模拟器 / 真机安装运行，验证应用正常启动、无异常
   五、单元测试编写规划
   测试框架
